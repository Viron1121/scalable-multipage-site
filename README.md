# Scalable Multi-Page Site

## Overview
A simple multi-page website demonstrating **clean structure, scalability, and responsive layout**.  
Includes a homepage (`index.html`) and a secondary page (`about.html`).

## How to Run
- **Locally:** Open any page in `src/pages/` in a browser, or use VS Code Live Server.  
- **Optional Deployment:** Can be hosted on GitHub Pages, Netlify, or Vercel.

## Adding New Pages
1. Add a new `.html` file in `src/pages/`.  
2. Include shared header and footer from `src/components`.  
3. Update navigation links if needed.  
4. Add page-specific CSS/JS as required.

## Responsive Design
Built mobile-first using CSS flexbox and media queries.
